# DJ Bot : A simple Discord Bot to play music in a guild.

## How to install :

### Optional - Create a virtualenv

* ```$ mkdir venv```
* ```$ virtualenv venv```
* ```$ source venv/bin/activate```

### Install the project

* ```$ python3 setup.py```

## Dependencies :

* discord.py[voice]
* google-api-python-client
* youtube_dl
* ffmpeg binary
